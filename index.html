Work

Former Research Intern at the Yale University Department of Hematology.

Over the summer preceding my senior year of high school, I worked under Dr. Manoj Pillai and in conjunction with computational biologist Dr. Anil Kesarwani to develop two bioinformatics tools using Python 3 for the analysis of patient genomes. Initially, I created a program to accept a genomic annotation file query (BED file) and a genomic sequence (FASTA file) that would output the desired portion of a sequence (FASTA file).

Genomic Sequence Searcher
Full Documentation

Cryptic Splice Site Identifier
Full Documentation

After developing this preliminary tool, I moved on to creating a program to accurately identify cryptic alternative splicing caused by mutated SF3B1, which has been linked to leukemia. My program worked by accepting a splice site documentation (GTF/GFF3 file) and outputting a tab-delimited output (STAR file) for further analysis. The output contained valuable information such as the number of uniquely mapped reads of each case, junction start and end points, as well as the statistical significance of each novel case.
