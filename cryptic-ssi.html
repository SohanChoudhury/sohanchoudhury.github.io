Cryptic Splice Site Identifier

For my the main project of my research internship at the Yale School of Medicine, I developed a program to identify novel cryptic splice sites in patient genomes.

Having completed the Genomic Sequence Searcher, I moved on to this more challenging project that directly adressed the research focus of the Pillai Lab.

This program took me approximately 8 weeks to complete. The purpose of it was to recognize impacts of a mutated SF3B1 protein on alternative splicing.

Given a GTF or GFF3 annotation file as well as a STAR documentation containing splice junctions, the Cryptic Splice Site Identifier will find all non-canonical splice sites.

Notable features:

  • Ability to use multiple .tab files
    • Option to input a .txt file containing paths of an unlimited number of .tab files
  • parser style user input
  • Versatile output file
    • Contains both cryptic splice junction and nearest annotated case
    • Provides distance of novel case from canonical case, as well as directionality
    • Tab delimited and contains number of uniquely mapped reads

The processing is very robust, as it works for 3', 5', or when both splice sites are cryptic.

This program was used by bioinformaticians at the lab to help direct research efforts by identifying statistically significant cases of cryptic splicing.

With this final project at the lab, I solidified my understanding of the field and my interest in Computer Science, amazed by the potential of such technologies.
